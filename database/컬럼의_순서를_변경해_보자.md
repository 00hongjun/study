# 컬럼의 순서를 변경해 보자

MySQL의 컬럼 순서를 변경하는 법에 대한 정리.


```sql

- 첫 위치로
ALTER TABLE 테이블명 MODIFY COLUMN 컬럼명 자료형 FIRST;

- 특정 컬럼 다음으로 
ALTER TABLE agency MODIFY COLUMN created_at datetime(6) AFTER use_sms;


- insert 시간 기록 
ALTER TABLE tableName MODIFY columnName TIMESTAMP DEFAULT CURRENT_TIMESTAMP NOT NULL;


- update 시간 기록
ALTER TABLE tableName MODIFY columnName TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP;

```
